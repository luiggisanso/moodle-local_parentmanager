## [1.0] - 2026-03-02

### Initial Release

- **Management Interface (CRUD):**
    - List view of all users with the Parent role.
    - Detail view showing children associated with a specific parent.
    - Ability to delete links (unassign role and clean up profile fields).
- **Basic CSV Import functionality.**
- **Manual Association Form:** Autocomplete fields to search for parents and children.
- **Cohort Association** page (`cohort.php`) dedicated to bulk assignment via cohorts.
- Role assignment in User Context.
- Basic Profile field population.
